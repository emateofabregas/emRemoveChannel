# emRemoveChannel
![emRemoveChannel_gizmo](https://github.com/user-attachments/assets/8cbdf14e-1d6f-40d3-a65e-87ba0e28e401)
Gizmo to remove of 100% beauty channel selected or just a percent
## Features:
* A checkbox to remove the specific channel selected (aov knob)
* Progressive remove knob to remove the AOV. 0 = no removed, 1 = removed.
* Connect a mask input to remove the AOV contribution in a specific area with its knobs.
